    # 加密算法和随机数

    将密钥或者密码保存到数据库中。 在加密算法的选择和使用上，有以下最佳实践：

    1. 不要使用ECB模式
    2. 不要使用流密码（RC4）
    3. 使用HMAC-SHA1代替MD5（甚至是代替SHA1）
    4. 不要使用相同的key做不同的事情
    5. salts与IV需要随机生成
    6. 不要自己实现加密算法，尽量使用安全专家已经实现好的库
    7. 不要依赖系统的保密性
