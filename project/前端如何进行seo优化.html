    # 语义化的HTML代码，符合W3C规范：语义化代码让搜索引擎容易理解网页

    1. 重要的内容HTML代码放在最前：搜索引擎抓取HTML顺序是从上到下的，保证重要内容一定会被抓取
    2. 重要内容不要用js输出：爬虫不会执行js获取内容
    3. 少用iframe：爬虫不会抓取iframe的内容
    4. 提高网站速度：网站速度是搜索引擎排序的一个重要指标
